# Anthropic Documentation Knowledge Base

This directory contains the latest documentation from Anthropic sources for use by agents.

## Documentation Structure

- `anthropic/` - General Anthropic engineering documentation
- `claude-code/` - Claude Code specific documentation
- `mcp/` - Model Context Protocol documentation
- `agents/` - Agent architecture and best practices
- `skills/` - Skills development documentation

## Auto-Update

This documentation is automatically updated by the hook system when:
- New frameworks or libraries are added to the project
- Documentation is older than 24 hours
- An agent is spawned that requires latest documentation
