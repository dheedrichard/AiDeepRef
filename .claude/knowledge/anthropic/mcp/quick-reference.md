# Model Context Protocol (MCP) Quick Reference

## Key Concepts

### What is MCP?
MCP enables Claude to interact with external tools and data sources through a standardized protocol.

### Core Components
- **Servers**: Provide tools and resources
- **Clients**: Connect to servers (Claude Code acts as a client)
- **Tools**: Functions that Claude can invoke
- **Resources**: Data that Claude can access

### Documentation Sources
- MCP Docs: https://modelcontextprotocol.io
- GitHub: https://github.com/modelcontextprotocol
- Examples: https://github.com/modelcontextprotocol/servers

## Common MCP Servers
- File system access
- Database connections
- API integrations
- Custom business logic
