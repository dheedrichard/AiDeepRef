# Shared Module

This directory contains reusable standalone components, directives, pipes, and models that are shared across multiple features.

## Structure

- **components/** - Reusable UI components (buttons, cards, modals, etc.)
- **directives/** - Custom directives
- **pipes/** - Custom pipes for data transformation
- **models/** - TypeScript interfaces and types used across the application

## Usage

Shared components are standalone and can be imported directly into any component that needs them.

## Examples

- RCS Badge component
- Video/Audio recorder components
- Transcript editor component
- Share modal component
- Notification drawer component
